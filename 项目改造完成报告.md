## 技术架构

### 数据流

```
用户操作 → 侧轮廓窗口 → 事件发送 → ThreeScene → 状态更新 → 模型重建
                                    ↓
                              sharedStore (全局状态)
                                    ↓
                              参数控制滑条 → 状态更新 → 模型重建
```

### 组件层级

```
App.vue
  └── ThreeScene.vue
        ├── RootSideProfile.vue (根侧轮廓窗口)
        ├── TipSideProfile.vue (尖侧轮廓窗口)
        └── 参数控制区域
              ├── XY Rotation 滑条
              └── A Tangent 滑条
```

### 状态管理

```
sharedPointsState (Pinia Store)
  ├── sideProfilePoints: G, H, I, J 及其控制点
  ├── xyRotation: 弧度值
  └── aTangent: 弧度值
```

## 关键技术点

### 1. 侧轮廓曲线实现

使用 `THREE.CubicBezierCurve3` 创建三次贝塞尔曲线：

```typescript
const curve = new THREE.CubicBezierCurve3(
  new THREE.Vector3(G.x, G.y, G.z),
  new THREE.Vector3(A_P1_G.x, A_P1_G.y, A_P1_G.z),
  new THREE.Vector3(A_P1_H.x, A_P1_H.y, A_P1_H.z),
  new THREE.Vector3(H.x, H.y, H.z)
)
```

通过 Vue 的事件系统和响应式系统实现：

```typescript

// 父组件监听事件

  // 更新本地状态
  // 同步到全局状态
```

### 3. 参数控制

### 功能操作

1. **调整侧轮廓**:
   - 在左侧"根侧轮廓 (G-H)"窗口中拖动G、H等控制点
   - 在右侧"尖侧轮廓 (I-J)"窗口中拖动I、J等控制点

   - 在任何3D窗口中使用鼠标左键旋转

## 故障排除

### 常见问题

1. **侧轮廓窗口不显示**:
   - 检查是否正确导入组件
   - 检查控制台是否有错误信息
   - 确认数据格式正确

2. **拖动无响应**:
   - 检查DragControls是否正确初始化
   - 确认相机和渲染器正常工作
   - 检查事件监听器是否正确绑定

3. **参数滑条无效**:
   - 检查变量绑定是否正确
   - 确认事件处理函数存在
   - 验证数据类型匹配
