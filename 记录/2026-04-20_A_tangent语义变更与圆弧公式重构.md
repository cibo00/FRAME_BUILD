## 一、改动背景

后端 `nailmodel.py` 中 `A_tangent` 参数的几何语义已切换为更直观的定义：

- **旧语义**：中间角 alpha，用于候选解公式 `R = span / (cos(alpha) - cos(alpha+phi))`
- **新语义**：`A_tangent = ∠(+Z, AD')`，即 A 点处 XZ 平面内 +Z 轴与 AD' 方向的夹角
- **默认值变更**：`π/10` (~18°) → `radians(86°)` (~86°)

圆弧求解链路从旧的候选解法改为三步新链路。

参考文档：飞书「A_tangent 语义变更与迁移指南」
参考代码：`C:\Users\cibo\Desktop\nailmodel.py` `_solve_ad_arc_from_tangent()` (lines 437-500)

### 2. 圆弧求解公式替换

**旧公式**（已删除）：
```
alpha = clamp(aTangent, [1e-4, π/2-1e-4])
phi_c = clamp(|AD_arc|, [1e-4, π-1e-4])
span = |AD|
denom = cos(alpha) - cos(alpha + phi_c)
R = span / denom
O = A + R*cos(alpha)*adUnit - R*sin(alpha)*Z_hat
```
带有回退到简单公式的 else 分支。

**新公式**（当前）：
```
aTangent = clamp(aTangent, [1e-4, π-1e-4])
phi = clamp(|AD_arc|, [1e-4, π-1e-4])
span_x = D.x - A.x

// Step 1: 用 A_tangent 和 AD x 跨度直接求 D'
d_prime_z = A.z + span_x * (cos(aTangent) / sin(aTangent))
D' = (D.x, 0, d_prime_z)

// Step 2: 弦 AD' 长度 + 圆心角 phi 求半径
chordLen = |D' - A|
R = chordLen / (2 * sin(phi/2))

// Step 3: 圆心在弦 AD' 的垂直平分线上
midpoint = (A + D') / 2
perp = (chord_z, 0, -chord_x)  // 弦在 XZ 平面的垂直方向
offset = chordLen / (2 * tan(phi/2))
O = midpoint + (offset / |perp|) * perp
```

关键区别：
- 旧公式把 A_tangent 当中间角 alpha，需要候选解筛选
- 新公式把 A_tangent 当作 D' 方向角，几何含义更直接
- A_tangent 裁剪范围从 `[1e-4, π/2]` 扩大到 `[1e-4, π]`
- 不再需要候选解 else 分支和回退公式

### 3. 滑条范围调整

A_tangent 滑条 `max` 从 `90` 改为 `180`，适配新默认值 86°。

## 五、侧轮廓曲线分段贝塞尔与拖拽约束（追加改动）

### 1. 问题

旧实现中根侧/尖侧轮廓各用一条 `CubicBezierCurve3`（4 控制点，三次），跳过了中间锚点 A/D：

| 曲线 | 后端定义（5 点，4 次） | 前端旧绘制（4 点，3 次） |
|---|---|---|
| 根侧 AGH | G → A_P1_G → **A** → A_P1_H → H | G → A_P1_G → A_P1_H → H（丢失 A） |
| 尖侧 JDI | J → D_P1_J → **D** → D_P1_I → I | J → D_P1_J → D_P1_I → I（丢失 D） |

### 2. 改为两段二次贝塞尔

每条侧轮廓曲线改为两段 `QuadraticBezierCurve3`（2 次贝塞尔），以锚点为衔接：

**根侧** (`RootSideProfile.vue` `createAGHCurve()`)：
- 段 1：G → A，控制点 A_P1_G
- 段 2：A → H，控制点 A_P1_H

**尖侧** (`TipSideProfile.vue` `createJDICurve()`)：
- 段 1：J → D，控制点 D_P1_J
- 段 2：D → I，控制点 D_P1_I

### 3. 控制点竖向拖拽锁定

控制点 A_P1_G、A_P1_H、D_P1_J、D_P1_I 只允许横向拖动（X 方向），Y 值锁定为初始值：

- `RootSideProfile.vue` drag 回调中：A_P1_G / A_P1_H 拖动后恢复 `position.y` 为 props 中的原始 Y
- `TipSideProfile.vue` drag 回调中：D_P1_J / D_P1_I 拖动后恢复 `position.y` 为 props 中的原始 Y

### 4. 涉及文件

- `src/components/RootSideProfile.vue` — `createAGHCurve()` 改为两段二次贝塞尔 + drag 锁 Y
- `src/components/TipSideProfile.vue` — `createJDICurve()` 改为两段二次贝塞尔 + drag 锁 Y
