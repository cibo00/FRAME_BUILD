## 一、改动背景

之前的实现方式是将后端传来的四元数（`cameraRotation`）应用到相机上旋转视角，相机围绕 `dataCenter` 轨道运动。现在需要改为：
- 四元数旋转应用到3D模型本身
- 相机固定从Z轴俯视（坐标原点）
- 滑条调整模型的旋转角度

## 二、核心改动

### 1. 新增 modelGroup 模型组 (`ThreeScene.vue`)

**声明与初始化：**
- 模块级声明 `let modelGroup: THREE.Group`
- `onMounted` 中创建并添加到 scene：
  ```ts
  modelGroup = new THREE.Group();
  scene.add(modelGroup);
  ```

**所有模型对象加入 modelGroup：**
- 球体 (spheres)、贝塞尔点 (bezier_points)
- 曲线 (curveObjects)、弧线 (adArc, AD_BF, AD_CE)
- 标签 (sphereLabels, bezierLabels)
- 指示线 (sphereLabelLines, bezierLabelLines)
- 填充点 (fillingPointsObject)
- 正则边 (A_end/D_end_regularized_edgeObject)
- `backgroundPlane` 保留在 scene 中，不随模型旋转

共计约 18 处 `scene.add` → `modelGroup.add`，8 处 `scene.remove` → `modelGroup.remove`

### 2. 旋转函数重写 (`ThreeScene.vue`)

**`applyCameraRotationFromArray()`** — 后端四元数应用到 modelGroup：
- 保留四元数/欧拉角计算逻辑
- `modelGroup.quaternion.copy(targetQuaternion)` 替代原来的相机旋转
- 删除相机位置计算、up向量更新、controls.target 设置、baseSpherical 计算
- 保留 localStorage 持久化
- 新增轴角差日志打印（相对于标准坐标轴）

**`applySlidersToCamera()`** — 滑条增量在世界坐标系下叠加：
```ts
// 分别绕世界坐标轴构建增量
const pitchQ = new THREE.Quaternion().setFromAxisAngle(
  new THREE.Vector3(1, 0, 0), pitchRad
);
const yawQ = new THREE.Quaternion().setFromAxisAngle(
  new THREE.Vector3(0, 1, 0), yawRad
);
const rollQ = new THREE.Quaternion().setFromAxisAngle(
  new THREE.Vector3(0, 0, 1), rollRad
);

// 世界坐标系叠加：delta * base
const targetQuaternion = deltaQ.clone().multiply(baseQ);
```
- 使用 `delta * base`（世界坐标系）替代原来的 `base * delta`（局部坐标系）
- 避免基准旋转到某些角度时 Y/Z 轴重合导致 roll 和 yaw 效果相同

### 3. 相机固定Z轴俯视 (`ThreeScene.vue`)

**初始化：**
```ts
camera.position.set(0, 0, 300);
camera.up.set(0, 1, 0);
camera.lookAt(0, 0, 0);
```

**首次 dataCenter 计算后：**
```ts
camera.position.set(0, 0, 300);
camera.lookAt(0, 0, 0);
controls.target.set(0, 0, 0);
```

- 删除 onMounted 中恢复 cameraPosition/cameraQuaternion 的代码
- localStorage 改为保存/恢复 `modelGroupQuaternion`

### 4. DragControls 适配 (`ThreeScene.vue`)

模型组旋转后，DragControls 设置的是世界坐标，需要转回局部坐标：
```ts
dragControls.addEventListener('drag', function (event) {
  if (modelGroup && event.object.parent === modelGroup) {
    const worldPos = new THREE.Vector3();
    event.object.getWorldPosition(worldPos);
    modelGroup.worldToLocal(worldPos);
    event.object.position.copy(worldPos);
  }
  // ...
});
```

## 三、背景图片闪烁修复 (`ThreeScene.vue`)

### 根因
`saveCameraDeltaForScene` / `loadCameraDeltaForScene` / `applySlidersToCamera` 写入 `props.sceneData.cameraDelta`，触发 `watch(() => props.sceneData, { deep: true })` 级联执行，导致 backgroundPlane 被 remove + textureLoader.load 重建。

### 修复
1. **移除所有对 `props.sceneData` 的写入**：
   - `saveCameraDeltaForScene` — 只存 localStorage
   - `loadCameraDeltaForScene` — 只从 localStorage 读取
   - `applySlidersToCamera` — 只调 saveCameraDeltaForScene
   - `resetCameraToProps` — 同理

2. **背景只在图片 URL 变化时重建**：
   ```ts
   const bgImageChanged = newData.backgroundImage !== lastLoadedBgImage;
   if (newData.backgroundImage && bgImageChanged) {
     // 才执行 textureLoader.load
   }
   ```

3. **删除 watcher 顶部无条件清理** backgroundPlane 的代码

4. **更新可拖拽列表时保留 backgroundPlane**：
   ```ts
   draggableObjects = [...spheres, ...bezier_points];
   if (allowBackgroundDrag.value && backgroundPlane && backgroundPlane.parent) {
     if (!draggableObjects.includes(backgroundPlane)) {
       draggableObjects.push(backgroundPlane);
     }
   }
   ```
