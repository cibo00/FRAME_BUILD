# 2026-04-21 扫掠曲面分段贝塞尔修复（点云不过 A/D'）

## 提交信息
- **分支**: `master`
- **改动文件**: `src/components/ThreeScene.vue`, `nailmodel.py`

---

## 一、问题现象

在尖侧/根侧轮廓窗口中拖动控制点（G/H/J/I/A_P1_G/A_P1_H/D_P1_J/D_P1_I）后，点云曲面不再经过 A 和 D' 点。调整 A_tangent 滑条时曲面始终有一点经过 A/D'，但拖动侧轮廓控制点后就不行了。

---

## 二、根因分析

### 侧轮廓窗口与扫掠函数的 Bezier 阶数不一致

侧轮廓窗口（`RootSideProfile.vue` / `TipSideProfile.vue`）在 2026-04-20 已改为**两段二次 Bezier**：

| 段 | 控制点 | 保证经过 |
|---|---|---|
| 根侧段 1 | G → A_P1_G → A | G 和 A |
| 根侧段 2 | A → A_P1_H → H | A 和 H |
| 尖侧段 1 | J → D_P1_J → D | J 和 D |
| 尖侧段 2 | D → D_P1_I → I | D 和 I |

A/D 作为两段曲线的端点，保证曲线经过。

但扫掠函数（前端 `ThreeScene.vue` 和 Python `nailmodel.py` 的 `_filling_surface`）仍然把 5 个控制点 `[G, A_P1_G, A, A_P1_H, H]` 当作**一条 4 阶 Bezier**。标准 Bezier 曲线只保证经过首尾端点（G 和 H），**不保证经过中间控制点 A**。

初始状态下 G/H 对称，Bezier 曲线的中点恰好在 A 上（对称性），所以看起来经过 A。拖动 G/H 打破对称性后，曲线不再经过 A，曲面也就不经过 A/D' 了。

---

## 三、修复方案

将扫掠函数中的单条 4 阶 Bezier 改为两段二次 Bezier，与侧轮廓窗口一致。

---

## 四、具体改动

### 改动 1：前端 `ThreeScene.vue`

**位置**: 约 2291-2296 行

**旧代码**:
```typescript
const sweepCurve = new BezierCurve(interpCtrl);
const sweepPts = sweepCurve.getPoints(_numPerCurve);
```

**新代码**:
```typescript
// 两段二次 Bezier（与侧轮廓窗口一致），保证曲面经过 A/D'
// interpCtrl 顺序: [G, A_P1_G, A, A_P1_H, H]
const halfPts = Math.max(Math.floor(_numPerCurve / 2), 2);
const curve1 = new BezierCurve([interpCtrl[0], interpCtrl[1], interpCtrl[2]]); // G → A_P1_G → A
const curve2 = new BezierCurve([interpCtrl[2], interpCtrl[3], interpCtrl[4]]); // A → A_P1_H → H
const sweepPts = [...curve1.getPoints(halfPts), ...curve2.getPoints(halfPts)];
```

### 改动 2：Python `nailmodel.py` — `_filling_surface`

**位置**: 约 671-680 行

**旧代码**:
```python
sweep_curve = Bezier(control_points)
curve_points = sweep_curve.get_points(num_points=num_points_per_curve)
```

**新代码**:
```python
# Two-segment quadratic Bezier (matches side-profile window rendering),
# ensuring the surface passes through A/D'.
# control_points order: [G, A_P1_G, A, A_P1_H, H]
half_pts = max(num_points_per_curve // 2, 2)
curve1 = Bezier(control_points[:3])   # G → A_P1_G → A
curve2 = Bezier(control_points[2:])   # A → A_P1_H → H
curve_points = np.vstack((
    curve1.get_points(num_points=half_pts),
    curve2.get_points(num_points=half_pts),
))
```

---

## 五、影响范围

- 两段二次 Bezier 在 A/D 点处 C0 连续（位置连续），与侧轮廓窗口行为一致
- 点云总采样点数不变（分成两半，每半各 `numPerCurve / 2` 个点）
- 曲面保证经过 A 和 D'（A/D 是两段 Bezier 的共享端点）
- Python 后端和前端行为完全一致
